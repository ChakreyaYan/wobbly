wobbly „Ä∞Ô∏è manages the state needed to calculate `x, y` rotations for a parallax effect, allowing you to focus the UI, and apply the effect how you want. It uses the [function as child](https://medium.com/merrickchristensen/function-as-child-components-5f3920a9ace9) and "prop getter" patterns, which gives you maximum flexibility with a minimal API.

## Installation

This module is distributed via [npm](https://www.npmjs.com/package/wobbly) which is bundled with [node](https://nodejs.org) and
should be installed as one of your project's `dependencies`:

```
npm install --save wobbly
```

> This package also depends on `react-vr`, `react` and `prop-types`. Please make sure you have those installed as well.
